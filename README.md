# go_api_rest

>Status do Projeto: Finalizado/Aguardando Melhorias

Quarto projeto utilizando a linguagem GO. 
Criação de uma API Rest 
O objetivo principal é aplicar os conceitos da linguagem e se aprofundar na estrutura, conexão e funcionamento do backend.


### Aplicados todos os conceitos do terceiro projeto com a linguagem:
- Reforçado o uso de variáveis, tipos e structs;
- Conversão de valores de variáveis utilizando strconv;
- Múltiplos retornos;
- Pacotes e visibilidade de pacotes;
- Modularização do código usando o padrão MVC;
- Integração com conhecimentos prévios; HTML, JavaScript e CSS;
- Integração com banco de dados utilizando PostgreSQL;
- Aplicados conceitos de direcionamento de rotas;
- Criação de modelos para trabalhar as funções do código de forma prática e organizada;
- Utilização de partials;

### Além da adição de conhecimentos como:
- Conexão com banco de dados utilizando Docker;
- Utilização de novas bibliotecas padrões do Go, aumentando a noção da linguagem;
- Entendendo o conceito do arquivo go.mod
- Trabalhando conceitos de retornos de Json e metodos HTTP;
- Utilizalção do GORM para facilitar a conexão entre o banco de dados e a aplicação;
- Criação de middleware para evitar código duplicado;
- Integração da API criado com um frontend React;
- Conhecimento acerca do termo CORS - Cross-Origin Resource Sharing;


## Para rodar o programa

1 - Tenha o GO instalado;

2 - Crie um banco de dados utilizando Docker;

    docker-composer up

3 - Certifique-se que os diretórios de apontamento e informações estão corretos;

4 - Inicie o terminal com:

    go run main.go

5 - Inicie a aplicação frontend com React;

    npm start

6 -  Be happy :smile:
